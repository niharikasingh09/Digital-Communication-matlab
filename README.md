# Digital Communication System Simulation using MATLAB

## Overview
This repository contains MATLAB simulations of digital communication systems.
The performance of BPSK and QPSK modulation schemes is analyzed over an AWGN channel.

## Implemented Schemes
- BPSK (Binary Phase Shift Keying)
- QPSK (Quadrature Phase Shift Keying)

## Objectives
- Implement modulation and demodulation techniques
- Analyze Bit Error Rate (BER) vs Eb/N0
- Compare simulated and theoretical performance

## Tools Used
- MATLAB

## Repository Structure
- BPSK/ : BPSK MATLAB code and output
- QPSK/ : QPSK MATLAB code and output
